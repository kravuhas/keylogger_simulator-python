Keylogger Simulator

📌 This project is a keylogger simulation script written in Python for educational purposes only.

⚠️ Disclaimer

This project is intended strictly for learning and testing.
Do not use it to monitor or collect data from others without explicit permission. Unauthorized use may be illegal.

📂 File Description
keylogger.py

This is the main script of the project. It is responsible for:

Listening to keyboard input events
Capturing keystrokes in real time
Storing the captured data locally for analysis/testing

The script demonstrates how keyboard event handling works in Python.

⚙️ Requirements

Make sure you have:

Python 3 installed

Required libraries (if any), for example:

pip install pynput
▶️ How to Run

Clone the repository:

git clone https://github.com/your-username/keylogger-simulator.git

Navigate to the folder:

cd keylogger-simulator

Run the script:

python keylogger.py
